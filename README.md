# 🖱️ Mouse Finder

마우스를 빠르게 **좌우로 흔들면** 커서가 macOS 스타일로 확대되는 Windows 유틸리티입니다.

---

## 실행 화면

마우스를 흔들면 커서가 순간적으로 커졌다가 부드럽게 원래 크기로 돌아옵니다.

---

## 기능

- **흔들기 감지** — 속도 + 방향 전환 횟수 + 최소 이동 거리(40px) 복합 조건으로 오발동 방지
- **macOS 스타일 애니메이션** — ease-out 빠른 확대 → 최대 유지 → ease-in-out 부드러운 복귀
- **커서 실시간 추적** — 애니메이션이 마우스를 따라다님, 실제 커서는 항상 위에 표시
- **주사율 자동 적응** — 60 / 120 / 144 Hz 모니터 자동 감지 후 프레임 간격 조정
- **클릭 투과** — 애니메이션 중에도 클릭이 아래 창으로 정상 전달
- **시스템 트레이** — 우클릭으로 설정 / 종료
- **감도 조절** — 슬라이더(10 ~ 100)로 흔들기 민감도 설정
- **Windows 자동실행** — 체크박스 한 번으로 시작프로그램 등록 / 해제
- **설정 영속화** — `config.json`에 저장, 다음 실행에도 유지

---

## 설치 및 실행

### 요구사항

- Windows 10 / 11
- Python 3.10 이상

### 방법 1 — Python으로 직접 실행

```powershell
python -m pip install pynput pystray Pillow
python main.py
```

### 방법 2 — 실행파일(.exe) 빌드

```powershell
.\build.ps1
# → dist\MouseFinder.exe 생성
```

빌드 후 `dist\MouseFinder.exe` 를 원하는 위치에 복사해 실행하면 됩니다.

---

## 사용법

1. 프로그램 실행 후 작업 표시줄 오른쪽 하단 **시스템 트레이** 아이콘 확인
2. 마우스를 빠르게 **좌우로 흔들기** → 커서 확대 애니메이션 표시
3. 트레이 아이콘 **우클릭 → 설정** 으로 감도 조절 및 자동실행 설정

| 설정 항목 | 설명 |
|---|---|
| 흔들기 감도 | 낮음(10) ~ 높음(100), 기본값 30 |
| Windows 자동실행 | 체크 시 로그인할 때 자동으로 시작 |

## 의존성

| 패키지 | 용도 |
|---|---|
| `pynput` | 마우스 이동 이벤트 수신 |
| `pystray` | 시스템 트레이 아이콘 |
| `Pillow` | 트레이 아이콘 이미지 생성 |
| `PyInstaller` | 실행파일 패키징 (빌드 시) |
